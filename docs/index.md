# Welcome to R-Type

R-type is a project that aims to reimagine the classic game "R-type" in C++ using a custom game engine and implementing networked multiplayer functionality.

## Features
- 💻 Cross-platform
- 🌐 Multiplayer
- 🧩 Custom Game Engine
- 📦 Easy to install
- 📚 Modern Documentation
